Proof of concept

ROC Curves

library(plotROC)
D.ex <- rbinom(100, 1, .5)

paired.data <- data.frame(M1 = rnorm(100, mean = D.ex), 
                          M2 = rnorm(100, mean = D.ex, sd = .4), 
                          M3 = runif(100), D = D.ex)

roc.ci <- calculate_roc(paired.data$M1, paired.data$D, ci = TRUE, alpha = 0.05)
ci.rocplot <- ggroc(roc.ci, label = "CI Example", ci = TRUE)

cat(
  export_interactive_roc(ci.rocplot, 
                         prefix = "aci", 
                         omit.d3 = TRUE)
)

Summary

  • Use rendering power of R -> create SVG for use in browser

  • Requires writing a bit of JS
  • Create R packages to accomplish specific task, e.g. plotROC
